## CPP-Console_Raycasting 
- это простой псевдо-3D движок на основе алгоритма RayCasting. Не использует сторонних библиотек, кроме WinAPI. 
Работают полностью в консоли. Поддерживает цветной вывод и различные разрешения экрана. Рекомендуются запускать в Windows Terminal.

## Возможности:
1. Цветной вывод
2. Огромные оптимизации и высокий FPS
3. Наличие оружия и врагов; возможность сражаться с врагами и умирать от них
4. BFS-поиск врагами игрока
5. Анимация выстрела пистолета
6. Звуки ходьбы и выстрела
7. И многое, многое другое!

## Сборка:
Нет никаких сторонних зависимостей, кроме WinAPI. Можно использовать как g++, так и компилировать через проект в Visual Studio.
